# Passo a passo

1. Criar a model `FlashCard`.
2. Gerar e registrar o adapter do Hive.
3. Abrir a box dos cards no `main.dart`.
4. Criar o `FlashCardRepository`.
5. Criar o `FlashCardStore`.
6. Passar o `deckId` para a tela de novo card.
7. Salvar `front` e `back` no botão `Adicionar`.
8. Buscar os cards do deck na tela de detalhe.
9. Mostrar a quantidade de cards no deck.
10. Usar a lista de cards para montar o quiz.
